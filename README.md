# C Logo

A logo for the C Programming Language, drawn with SVG using JavaScript.

## Development

```console
$ git clone git@github.com:polyglotprog/c-logo.git
$ cd c-logo
$ npm install
$ npm run dev
```

## Building

```console
$ npm run build
```
